## Выполненные базовая эксплуатация Domain Controller:

### 1. Установка роли AD DS
- Через **Server Manager** установлена роль:
- Active Directory Domain Services (AD DS)
- Необходимые дополнительные компоненты установлены автоматически


### 2. Повышение сервера до контроллера домена
- Запущен мастер **Promote this server to a domain controller**
- Выбрана опция **Add a new forest**
- Создан новый лес Active Directory

#### Параметры домена
- Имя корневого домена: `corp.local`
- NetBIOS-имя: `CORP`
- DNS-сервер: установлен совместно с AD DS
- Global Catalog: включён
- Режим работы домена и леса: Windows Server 2016
- Задан пароль: DSRM

После завершения установки сервер был автоматически перезагружен

## Состояние после установки
- Сервер выполняет роль **Domain Controller**
- DNS-сервер стал авторитетным для зоны `сorp.local`
- Вход в систему осуществляется под доменной учетной записью: CORP\Administrator


### Клиентская машина (Windows 10)
- Клиентская ОС: Windows 10 Pro
