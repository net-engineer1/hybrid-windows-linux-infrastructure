## Hybrid Windows + Linux Infrastructure

Пет-проект по развертыванию и администрированию гибридной инфраструктуры Windows + Linux: Active Directory, сети, мониторинг, резервное копирование и автоматизация.

## Цели проекта

- Получить практические навыки системного администрирования.
- Научиться работать с Windows Server и Linux.
- Построить тестовую корпоративную инфраструктуру.
- Вести документацию и историю изменений через Git.

## Используемые технологии

- Windows Server
- Linux (Ubuntu)
- Active Directory, DNS, DHCP
- Git / GitHub
- VirtualBox
